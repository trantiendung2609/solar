Solar

Solar is a fully decentralized and open-source blockchain designed from core blockchain components. It operates via a Delegated Proof of Stake consensus and secured by 53 voted-in delegates that produce blocks and validate transactions.

Become a delegate or a voter

Participate

Having the right forging delegate is important for the network. Usuallyeach public delegate offers share in rewards, sometimes in combinationwith additional contribution to the network or project. A delegate cancontribute to the security of the network or contribute towards commarketing, community product development, which could returnin term or long term a better value for the asset and project.

Features that will enhance your project

Build your own dApps

With Solar you can create your own NFT's, dApps or fungible tokens. The Solar Blockchain provides the end-user with rich libraries and tools develop projects on the blockchain for many usecases. As an example, these tools will allow you to publish NFT's in the form of text, images, audio or video's on the blockchain using our SLP library. SLP stands for Side Ledger Protocol. A sidechain with custom transactions that also enables the user to create fungible tokens in a few simple clicks directly from within the wallet. All of these features are provided with documentation and easy to use Rest API's that can be implemented or integrated in other 3rd party tools and applications.
